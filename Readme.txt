Для работы проекта использовались библиотеки flask и request.

База данных - sqlite3. Она автоматически создаётся в папке db при запуске веб сервиса.
Все методы, относящиеся к БД находятся в файле db/names_bd.py
Для заполнения таблицы names с ФИО пользователя используются данные из файла db/names.txt
При каждом новом запуске веб сервиса, таблица names пересоздаётся заново.

Методы веб сервиса находятся в файле web_service/web_service.py
Получить ФИО пользователя по его id можно с помощью POST запроса с заголовком Content-Type: application/x-www-form-urlencoded и телом id=%номер_пользователя%

Все методы автотестов находятся в файле test/autotests.py
Тестовые данные, использующиеся для обращения к веб сервису беруться из списка, возвращаемого методов getData, в файле test/test_data.py

Логи тестов выводятся на консоль и записываются в файл test/log.txt, который создаётся, если его не существует и перезаписывается при каждом запуске автотестов
