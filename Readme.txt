Для работы проекта использовались библиотеки flask и request.

База данных - sqlite3. Все методы, относящиеся к БД находятся в файле names_bd.py
Для заполнения таблицы names с ФИО пользователя используются данные из файла names.txt
При каждом новом запуске веб сервиса, таблица names пересоздаётся заново.

Методы веб сервиса находятся в файле web_service.py
Получить ФИО пользователя по его id можно с помощью POST запроса с заголовком Content-Type: application/x-www-form-urlencoded и телом id=%номер_пользователя%

Все методы автотестов находятся в файле autotests.py
Тестовые данные, использующиеся для обращения к веб сервису беруться из списка, возвращаемого методов getData, в файле test_data.py

Логи тестов выводятся на консоль и записываются в файл log.txt
